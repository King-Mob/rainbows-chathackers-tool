# Rainbow Chathackers Module

Rainbow module for Chathackers Matrix chat integration tools.

This app creates rainbows within a group chat. It has a keyword that the app responds to ("rainbow") and then the rest of the interface is put together from emoji reactions and replies.

## Getting started

1. Clone the repo
1. Run the command `npm install`
1. Run the command `npm run dev`
1. Copy the `rainbow.json` file and paste into the `registrations` folder of [the wrapper project](https://github.com/King-Mob/chathackers-wrapper)
1. Ask your testing buddy to send "rainbow" to the WhatsApp group
1. Marvel in delight at your newfound power
